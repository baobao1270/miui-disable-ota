# MIUI Disable OTA
通过修改 system.prop 中版本号的方式屏蔽 OTA 检测更新，同时**禁用系统更新 App**。

可能有未知的副作用。

## 使用方式
**不要直接下载 Release，否则可能变砖！！！** Release 里的文件是给我自己用的。

1. `clone 本仓库，按照以下方法修改 `src/system.prop`
2. 确保手机已经安装 Magisk。
3. 将手机连接到计算机，打开 ADB 调试模式，并至少执行过 adb shell 一次以获得信任。打开文件传输模式。
4. 运行 build.sh。需要计算机上安装 zip 命令。
5. 在 Magisk 中刷入内部存储根目录的 `miui-disable-ota.zip` 文件。重启，进入系统设置查看版本号以确认生效。

### MIUI 12
按照版本号规则，直接设置一个比较高的版本号即可，如开发版 `12.99.7.12`，正式版 `12.99.99.99`

### MIUI 13
必须为当前最新版本号。
